2. Planning

### Vocabulary cheat sheet

- Hero: the first visible section at the top of the page
- CTA: call to action, the main button or next step
- Above the fold: what users see before scrolling
- Visual hierarchy: what grabs attention first, second, third
- Card: a boxed content block
- Section spacing: the vertical breathing room between sections
- Responsive: layout adapts to mobile, tablet, desktop
- Accent color: the main highlight color used sparingly
