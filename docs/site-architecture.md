# Site Architecture

## Objective

Create a professional portfolio that works for both:

- recruiters / hiring managers
- clients / founders / business owners

## Core Sections

### Hero

- name
- positioning statement
- short value proposition
- primary CTA
- secondary CTA

### Credibility Strip

- selected metrics
- roles
- technologies
- promise statements

### Selected Work

- strongest 3 projects first
- problem
- role
- outcome
- stack

### Capabilities

- what you can do for clients / teams
- technical areas
- delivery areas

### Experience

- timeline or highlights
- company / project
- role
- impact

### About

- short personal summary
- working style
- what you care about

### Contact

- email
- social links
- resume link
- strong closing CTA

## Visual Direction

- dark editorial base
- warm neutral highlights instead of purple
- oversized typography
- crisp geometric framing
- subtle gradients and texture
- deliberate motion, not constant motion

## UX Rules

- clear CTA above the fold
- readable in under 60 seconds
- mobile-first responsiveness
- section spacing must feel calm and premium
- no section should exist only for decoration
